# ミルマンの定理 - Millman's theorem

内部抵抗を考慮した、電池の並列接続についての公式。

![ミルマンの定理の回路図](https://detail-infomation.com/wp-content/uploads/2021/12/13c5e30cd3fb7cc92bec030e1b2a4be6-700x524.png)

（[Electrical Information](https://detail-infomation.com/millmans-theorem/) より）

合成起電力は、**“コンダクタンス（抵抗の逆数）を重みとした”各起電力の重み付き平均**になる。

```math
V = \frac{\sum_i \frac{V_i}{R_i}}{\sum_i \frac{1}{R_i}} = \frac{\sum_i G_i V_i}{\sum_i G_i}
```

よくある抵抗を用いた記述だと難解だが、重み付き平均として捉えると非常に分かりやすい。
## 証明

テブナンの定理より、端子を開放した状況を仮定して、回路片全体の起電力を求めればよい。
各電池に流れる電流 $I_i$ は以下で表される。

```math
I_i = -\frac{V - V_i}{R_i} = G_i (V_i - V)
```

キルヒホッフの第一法則より、電流の和は $0$ になる。

```math
\sum_i G_i (V_i - V) = 0
```

（余談だが、推定方程式そっくりの形。）

これを解けば

```math
V = \frac{\sum_i G_i V_i}{\sum_i G_i}
```

が得られる。
