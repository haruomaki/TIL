# 新規プロジェクト作成手順

## Vanilla TypeScript

```bash
❯ npm create vite@latest .

> npx
> "create-vite" .

│
◇  Select a framework:
│  Vanilla
│
◇  Select a variant:
│  TypeScript
│
◇  Use rolldown-vite (Experimental)?:
│  No
│
◇  Install with npm and start now?
│  Yes
```

適宜 `package.json` を編集してプロジェクト名などを変更する。

```bash
npm install -D nanohtml
npm install -D unplugin-auto-import
```
